03 // projects

Selected projects

security-in-computing

Security
Goal

Understand how classical encryption actually works by building it, not importing it, for a foundations-of-security project.

Approach

Implemented the Affine, Playfair, and Vigenère ciphers from scratch in Python, with no crypto libraries. Wrote the encrypt and decrypt logic by hand.

Outcome

Got a real feel for how substitution and polyalphabetic ciphers work, and where they fall apart. This is the project that made me want to go into security properly.
